🎯 Révision du concept créatif
Mirepoix = bastide médiévale, couverts en bois, halle historique, ambiance chaleureuse et traditionnelle du Sud-Ouest.
Le concept pivote vers : "Éditorial Sud-Ouest / Bistronomie de bastide"

Un site qui conjugue l'âme des vieilles pierres médiévales et la générosité de la table du Sud-Ouest. Pas sombre-cinéma, plutôt chaleureux-authentique, lumineux-terroir, avec une touche éditoriale moderne.

⚡ Effets & micro-interactions "signature"

Curseur custom sur desktop : petit point terre-cuite qui s'élargit sur les liens
Scroll reveal lent (GSAP + Lenis) : les textes éditoriaux montent avec fade
Dropcaps animées : la capitale se dessine au scroll
Grain papier en overlay permanent (très léger)
Hover images : léger zoom + liseré doré qui se trace
Filets dorés animés entre sections (se tracent de gauche à droite au scroll)
Transitions de page : fondu crème (overlay parchemin qui couvre/découvre)
Hero diaporama : fondu lent entre 3 photos d'ambiance
Bouton "Réserver" flottant mobile (sticky bottom)


🧱 Stack technique recommandée

Thème custom : base Sage 10 (Roots) ou starter Vite + Tailwind
ACF Pro : répéteurs pour carte, producteurs, galerie
GSAP + ScrollTrigger + Lenis : animations et smooth scroll
Swiper.js : slider signatures + hero
Mapbox GL JS : carte custom (style parchemin)
Images : conversion WebP systématique, <picture> avec srcset, lazy loading natif
Fonts : font-display: swap, preload Fraunces variable
Perf : critical CSS inline, Lighthouse cible > 90
